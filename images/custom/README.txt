Omni Custom Pack
Follow steps below:

1. Put Images into "custom" folder
2. Double click "OmniPackGen.exe"
 2a. This .exe will scan the current folder for all images and create a .txt file listing all image names for Omni to use
3. Wait for it to finish running (Finished when "customPack.txt" appears)
4. Enable Custom slider in Omni
5. Enjoy

"Repeat process for any additional changes to the folder"
